# Customer Segmentation Using Unsupervised Learning

## Objective

The objective of this project is to segment customers based on their
income and spending behavior using K-Means clustering. This helps in
understanding different customer groups and designing targeted marketing
strategies.

## Dataset

Mall Customers Dataset\
https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python

## Steps Performed

1.  Data loading and preprocessing\
2.  Exploratory Data Analysis (EDA)\
3.  Feature selection (Income & Spending Score)\
4.  Data scaling using StandardScaler\
5.  Finding optimal clusters using Elbow Method\
6.  Applying K-Means clustering\
7.  Visualization using scatter plots and PCA\
8.  Customer segmentation and labeling

## Customer Segments & Strategies

-   Premium Customers: High income, high spending → Target with luxury
    products\
-   Average Customers: Moderate income & spending → Promote mid-range
    products\
-   Low Income High Spenders: Offer discounts and loyalty programs\
-   Rich but Low Spenders: Encourage spending with exclusive offers\
-   Budget Customers: Provide budget-friendly deals

## Skills Gained

-   Unsupervised Learning (K-Means)\
-   Dimensionality Reduction (PCA)\
-   Data Visualization\
-   Customer Segmentation\
-   Business Strategy Development

## Conclusion

This project successfully segmented customers into meaningful groups
using K-Means clustering. These insights can help businesses apply
targeted marketing strategies, improve customer satisfaction, and
increase revenue.
